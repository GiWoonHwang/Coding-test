변수란 데이터를 저장하기 위해 프로그램에 의해 이름을 할당받은 메모리 공간을 의미
즉, 변수란 데이터를 저장할 수 잇는 메모리 공간을 의미하며, 이렇게 저장된 값은 변경될 수 있습니다.

c++에서 숫자 표현에 관련된 변수는 정수형, 실수형으로 구분할 수 있다.
또다시 정수형 변수는 -> char, int, long, longlong으로 실수형 변수 -> float, double로 구분된다.
관련된 데이터를 한 번에 묶어서 처리하는 사용자 정의 구조체 변수도 있다.

변수의 이름 생성 규칙
c++에서는 변수의 이름을 비교적 자유롭게 지을 수 있다.
변수의 이름은 해당 변수에 저장될 데이터의 의미를 잘 나타내도록 짓는 것이 좋다.

c++에서 변수의 이름을 생성할 때 반드시 지켜야 하는 규칙은 다음과 같습니다.
1. 변수의 이름은 영문자, 숫자, 언더스코어로만 구성됩니다.
2. 변수의 이름은 숫자로 시작될 수 없다.
3. 변수의 이름 사이에 공백을 포함할 수 없다.
4. 변수의 이름으로 c++에서 미리 정의된 키워드는 사용 될 수 없다.
5. 변수 이름의 길이에는 제한이 없다.
(c++에서는 변수의 이름에 대소문자를 구분하므로 이 점에 주의해야 한다)

비트와 바이트
컴퓨터는 모든 데이터를 2진수로 표현하고 처리합니다.
비트란 컴퓨터가 데이터를 처리하기 위해 사용하는 데이터의 최소 단위 입니다.
이러한 비트에는 2진수의 값(0과1)을 단 하나만 저장할 수 있습니다.
바이트란 위과 같은 비트가 8개 모여서 구성되며, 한 문자를 표현할 수 있는 최소 단위

변수와 메모리 주소
변수는 기본적으로 메모리 주소를 기억하는 역할을 한다.
메모리 주소란 물리적인 메모리 공간을 서로 구분하기 위해 사용되는 일종의 식별자
즉 메모리 주소란 메모리 공간에서의 정확한 위치를 식별하기 위한 고유 주소를 의미를

변수를 참조할 때는 메모리의 주소를 참조하는 것이 아닌, 해당 주소에 저장된 데이터를 참조하게 된다
따라서 변수는 데이터가 저장된 메모리의 주소뿐만 아니라, 저장된 데이터의 길이와 형태에 관한 정보도 같이 기억해야 한다.

변수의 선언
c++에서는 변수를 사용하기 전에 반드시 먼저 해당 변수를 저장하기 위한 메모리 공간을 할당받아야 한다.
1. 변수의 선언만 하는 방법
    변수를 먼저 선언하여 메모리 공간을 할당받고, 나중에 변수를 초기화 한다.
    이렇게 선언된 변수는 초기화 하지 않았기 때문에 해당 메모리 공간에는 알 수 없는 쓰레기값만이 들어가 있다.
    따라서 초기화 하지 않은 변수는 절대 사용해서는 안된다.

2. 변수의 선언과 동시에 초기화 하는 방법
    변수의 선언과 동시에 그 값을 초기화할 수 있다.
    선언하고자 하는 변수들의 타입이 같다면 이를 동시에 선언할 수 있다.
